# Calibration Guide

This document explains how to calibrate pedal travel using the UI and the no‑UI fallback gesture.

## UI Calibration (Recommended)

1. Release pedal → click **Capture Min**.
2. Press pedal fully → click **Capture Max**.
3. Click **Save to Device**.

## No‑UI Boot Gesture (Fallback)

- After power‑up, press the pedal fully **three times quickly** (within ~5 seconds).
- The firmware enters calibration mode and captures MIN and MAX automatically.

## Expected Results

- Rest position → about **0.0 %**
- Full press → about **100.0 %**

## Tips

- Adjust mechanical stops **before** calibration.
- Repeat calibration if hardware changes.
