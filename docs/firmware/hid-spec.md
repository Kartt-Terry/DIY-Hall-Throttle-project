# USB HID Specification (Throttle Axis)

This document defines the HID output format used by the Arduino Pro Micro.

- Report type: joystick or gamepad axis
- Resolution: 10–16 bits (default 10 bits)
- Target update rate: 250–1000 Hz

Note: ATmega32U4 supports HID natively. Libraries such as Arduino Joystick or NicoHood HID may be used later.
