# UI Modes (Sidebar Navigation)

- **Driving** — Live view (percentage, graph, active profile, status)
- **Settings** — Adjust profile (Linear/Expo/Log), expo value, low/high deadzones, run calibration (Capture Min/Max, Save)
- **Fault** — Red banner, error code + description, suggested actions, reset button
- **Diagnostics** — Raw/Filtered/Output values, noise (std), update rate, I²C errors, firmware version, uptime
